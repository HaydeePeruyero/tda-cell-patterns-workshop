1 Análisis de diferencias entre grupos biológicos mediante TopKAT

Hasta ahora hemos utilizado la homología persistente para describir la organización espacial de las células y hemos comparado muestras mediante distancias entre diagramas de persistencia.

Sin embargo, una pregunta biológica importante permanece abierta:

¿Las diferencias topológicas observadas entre las muestras están asociadas con variables clínicas o biológicas de interés?

Por ejemplo, en nuestro conjunto de datos nos interesa determinar si la organización espacial de las células difiere entre pacientes con anemia de Fanconi (FA) y pacientes sin anemia de Fanconi (No-FA), así como estudiar si estas diferencias dependen del tipo de tejido analizado (carcinoma, displasia o estroma).

Para responder este tipo de preguntas utilizaremos TopKAT (Topological Kernel Association Test), una metodología que combina herramientas de análisis topológico de datos (TDA) con pruebas estadísticas de asociación basadas en kernels. Esta herramienta fue desarrollada por Sarah Samorodnitsky y colaboradores para analizar estructuras topológicas presentes en datos espaciales y evaluar formalmente su relación con variables clínicas o biológicas.

Para obtener más información sobre la metodología, documentación y aplicaciones, puede consultarse el sitio oficial:

TopKAT Project Page

1.1 ¿Qué hace TopKAT?

De manera general, TopKAT sigue el siguiente flujo de trabajo:

Coordenadas celulares
          ↓
Complejo de Vietoris–Rips
          ↓
Homología persistente
          ↓
Diagramas de persistencia
          ↓
Matriz de similitud topológica
          ↓
TopKAT
          ↓
Prueba estadística de asociación

La idea fundamental consiste en transformar cada muestra en un conjunto de características topológicas mediante homología persistente y posteriormente evaluar si las muestras que presentan topologías similares también comparten características biológicas o clínicas similares.

A diferencia de los análisis realizados anteriormente, cuyo objetivo principal era describir o visualizar similitudes entre muestras, TopKAT permite realizar inferencia estadística y responder preguntas como:

  • ¿Existen diferencias topológicas significativas entre muestras FA y No-FA?
  • ¿Las diferencias observadas ocurren en componentes conexas ((H_0)) o en ciclos ((H_1))?
  • ¿A qué escala espacial aparecen las mayores diferencias entre grupos?
  • ¿Hay diferencias entre las interacciones celulares entre muestras FA y No-FA?

1.2 Aplicación en este estudio

En este taller utilizaremos TopKAT para comparar muestras provenientes de pacientes con anemia de Fanconi y pacientes sin anemia de Fanconi.

Las comparaciones se realizarán de manera independiente para distintos compartimentos histológicos:

  • Carcinoma.
  • Displasia.
  • Estroma.

Nuestro objetivo será determinar si existen diferencias significativas en la organización topológica de las células entre ambos grupos y explorar la escala espacial en la que dichas diferencias son más evidentes.

En las siguientes secciones construiremos las matrices de similitud requeridas por TopKAT, ejecutaremos las pruebas de asociación y analizaremos la interpretación biológica de los resultados obtenidos.

2.5 Cálculo de matrices de similitud topológica

Después de visualizar las distribuciones espaciales de las células, el siguiente paso es transformar cada muestra en una representación topológica que capture propiedades globales de su organización espacial.

Para ello utilizamos la función rips_similarity_matrix() del paquete TopKAT. Esta función automatiza gran parte del flujo de trabajo de TDA y realiza las siguientes tareas:

  1. Construye una filtración de complejos de Vietoris–Rips para cada muestra.
  2. Genera los diagramas de persistencia correspondientes.
  3. Genera matrices kernel que cuantifican la similitud topológica entre pares de muestras.

2.6 Construcción de matrices kernel topológicas

Una vez obtenidos los diagramas de persistencia, al igual que las secciones anteriores, necesitamos una forma de comparar las muestras entre sí.

TopKAT resuelve este problema mediante funciones kernel, que transforman la información topológica en matrices de similitud. Estas matrices resumen la información contenida en los diagramas de persistencia y constituyen la entrada principal para el análisis estadístico posterior.

El primer kernel utiliza información relacionada con la aparición y fusión de componentes conexas durante la filtración.

2.8 Prueba de asociación topológica con TopKAT

Una vez obtenidas las matrices kernel, podemos evaluar si la organización espacial de las células difiere entre los grupos estudiados. Para ello utilizamos Topological Kernel Association Test (TopKAT). En este ejemplo, la variable de respuesta corresponde al grupo al que pertenece cada muestra (FA o No FA) definida anteriormente.

Mostrar código 
res <- TopKAT(
  y = y,
  X = NULL,
  K.list = simmat$K.list,
  omega.list = c(0, 0.5, 1),
  outcome.type = "binary"
)

Hipótesis evaluadas

TopKAT contrasta las siguientes hipótesis:

  • **Hipótesis nula: la estructura topológica es similar entre los grupos.
  • **Hipótesis alternativa: existen diferencias topológicas entre los grupos.

Valor-p global

El resultado principal del análisis corresponde al valor-p global:

Mostrar código 
res$overall.pval
TipOutput

res$overall.pval [1] 3.308889e-05

Este valor resume la evidencia estadística de asociación entre la topología de las muestras y el grupo al que pertenecen.

De manera general:

  • Valores pequeños de p (en este caso si (p < 0.05) ), existe evidencia estadísticamente significativa de diferencias topológicas entre FA y NoFA.
  • Valores grandes de p (en este caso si (p ) ), no existe evidencia suficiente para rechazar la hipótesis nula.

Contribución de cada dimensión topológica

Además del valor-p global, TopKAT reporta resultados individuales para cada kernel utilizado en el análisis.

Mostrar código

TipOutput

2.9 Identificación de escalas espaciales relevantes

Hasta este punto hemos evaluado si existen diferencias topológicas entre los grupos. Sin embargo, también es interesante identificar a qué escala espacial resaltan esas diferencias.

La función scale_importance() permite estudiar cómo cambia la asociación topológica al considerar distintos valores de (). Para ello, el procedimiento recorre una secuencia de escalas y calcula nuevamente la evidencia estadística en cada una de ellas.

Mostrar código 
res_scale_import <- scale_importance(
  pd.list = simmat$rips.list,
  y = y,
  omega.list = c(0, 0.5, 1),
  threshold = 50,
  PIDs = 1:100,
  outcome.type = "binary"
)

Organización de los resultados

Los valores obtenidos pueden almacenarse en un dataframe para facilitar su exploración y visualización.

Mostrar código 
res_scale_import.df <- data.frame(
  thresh = res_scale_import$threshold.seq,
  pval = res_scale_import$pvals
)

Escala de mayor relevancia

La función también identifica automáticamente la escala asociada al valor-p mínimo.

2.12 Conectividad promedio entre grupos FA y NoFA

Hasta este punto hemos explorado muestras individuales. Sin embargo, también resulta útil resumir la organización espacial celular característica de cada grupo clínico y asi podemos ver si existe algún comportamiento en común entre las muestras de un grupo.

Para ello, se construyeron matrices de conectividad promedio para los grupos FA y NoFA utilizando la escala óptima identificada previamente. Estas matrices resumen la frecuencia con la que distintos tipos celulares aparecen conectados dentro de cada condición.

Mostrar código 
# ============================================================
# Conectividad promedio FA vs NoFA
# ============================================================

library(dplyr)
library(stringr)
library(reshape2)
library(ggplot2)
library(viridis)

# Un PID por archivo
pid_df <- data1.df %>%
  select(PID, archivo) %>%
  distinct()

# Grupo: FA = 1, NoFA = 0
pid_df$grupo <- ifelse(
  grepl("F", pid_df$archivo, ignore.case = TRUE),
  1,
  0
)

table(pid_df$grupo)

cell.types <- sort(unique(data1.df$type))
threshold <- res_scale_import$min.thresh

# Matrices acumulativas
connect_FA <- matrix(
  0,
  nrow = length(cell.types),
  ncol = length(cell.types),
  dimnames = list(cell.types, cell.types)
)

connect_noFA <- connect_FA

# Calcular conectividad para cada PID
for(i in seq_len(nrow(pid_df))) {

  pid <- pid_df$PID[i]
  grupo <- pid_df$grupo[i]

  patient <- data1.df %>%
    filter(PID == pid)

  connect_i <- generate_connectivity(
    images.df = patient,
    threshold = threshold,
    type.column = "type",
    unique.types = cell.types
  )

  tmp <- matrix(
    0,
    nrow = length(cell.types),
    ncol = length(cell.types),
    dimnames = list(cell.types, cell.types)
  )

  tmp[
    rownames(connect_i),
    colnames(connect_i)
  ] <- connect_i

  if(grupo == 1){
    connect_FA <- connect_FA + tmp
  } else {
    connect_noFA <- connect_noFA + tmp
  }
}

# Promedios
n_FA <- sum(pid_df$grupo == 1)
n_noFA <- sum(pid_df$grupo == 0)

connect_FA <- connect_FA / n_FA
connect_noFA <- connect_noFA / n_noFA

plot_connectivity_matrix <- function(mat, titulo){

  ggplot(
    reshape2::melt(mat),
    aes(Var1, Var2, fill = value)
  ) +
    geom_tile(color = "white") +
    viridis::scale_fill_viridis(option = "turbo") +
    theme_bw() +
    coord_equal() +
    labs(
      x = "Cell Type",
      y = "Cell Type",
      fill = "Average\nConnections",
      title = titulo
    ) +
    theme(
      axis.text.x = element_text(
        angle = 45,
        hjust = 1
      )
    )
}

p_FA <- plot_connectivity_matrix(
  connect_FA,
  paste0("Average Connectivity (FA, n=", n_FA, ")")
)

p_noFA <- plot_connectivity_matrix(
  connect_noFA,
  paste0("Average Connectivity (No FA, n=", n_noFA, ")")
)

p_FA + p_noFA
TipOutput

Para cada muestra se construye una matriz de conectividad entre tipos celulares. Posteriormente, las matrices individuales se agrupan según la condición clínica (FA o NoFA) y se promedian para obtener una representación global de cada grupo. Los mapas de calor permiten identificar qué pares de tipos celulares presentan una mayor conectividad promedio. Comparar ambas matrices puede revelar diferencias en la organización espacial del microambiente tumoral que no son evidentes al analizar muestras individuales de forma aislada.

# Análisis de diferencias entre grupos biológicos mediante TopKAT

Hasta ahora hemos utilizado la homología persistente para describir la organización espacial de las células y hemos comparado muestras mediante distancias entre diagramas de persistencia.

Sin embargo, una pregunta biológica importante permanece abierta:

> ¿Las diferencias topológicas observadas entre las muestras están asociadas con variables clínicas o biológicas de interés?

Por ejemplo, en nuestro conjunto de datos nos interesa determinar si la organización espacial de las células difiere entre pacientes con anemia de Fanconi (FA) y pacientes sin anemia de Fanconi (No-FA), así como estudiar si estas diferencias dependen del tipo de tejido analizado (carcinoma, displasia o estroma).

Para responder este tipo de preguntas utilizaremos **TopKAT** (*Topological Kernel Association Test*), una metodología que combina herramientas de análisis topológico de datos (TDA) con pruebas estadísticas de asociación basadas en kernels. Esta herramienta fue desarrollada por Sarah Samorodnitsky y colaboradores para analizar estructuras topológicas presentes en datos espaciales y evaluar formalmente su relación con variables clínicas o biológicas.

Para obtener más información sobre la metodología, documentación y aplicaciones, puede consultarse el sitio oficial:

[TopKAT Project Page](https://sarahsamorodnitsky.github.io/TopKAT/)

## ¿Qué hace TopKAT?

De manera general, TopKAT sigue el siguiente flujo de trabajo:

``` text
Coordenadas celulares
          ↓
Complejo de Vietoris–Rips
          ↓
Homología persistente
          ↓
Diagramas de persistencia
          ↓
Matriz de similitud topológica
          ↓
TopKAT
          ↓
Prueba estadística de asociación
```

La idea fundamental consiste en transformar cada muestra en un conjunto de características topológicas mediante homología persistente y posteriormente evaluar si las muestras que presentan topologías similares también comparten características biológicas o clínicas similares.

A diferencia de los análisis realizados anteriormente, cuyo objetivo principal era describir o visualizar similitudes entre muestras, TopKAT permite realizar inferencia estadística y responder preguntas como:

- ¿Existen diferencias topológicas significativas entre muestras FA y No-FA?
- ¿Las diferencias observadas ocurren en componentes conexas ((H_0)) o en ciclos ((H_1))?
- ¿A qué escala espacial aparecen las mayores diferencias entre grupos?
- ¿Hay diferencias entre las interacciones celulares entre muestras FA y No-FA?

## Aplicación en este estudio

En este taller utilizaremos TopKAT para comparar muestras provenientes de pacientes con anemia de Fanconi y pacientes sin anemia de Fanconi.

Las comparaciones se realizarán de manera independiente para distintos compartimentos histológicos:

- Carcinoma.
- Displasia.
- Estroma.

Nuestro objetivo será determinar si existen diferencias significativas en la organización topológica de las células entre ambos grupos y explorar la escala espacial en la que dichas diferencias son más evidentes.

En las siguientes secciones construiremos las matrices de similitud requeridas por TopKAT, ejecutaremos las pruebas de asociación y analizaremos la interpretación biológica de los resultados obtenidos.

## Prueba de asociación topológica con TopKAT

Una vez obtenidas las matrices kernel, podemos evaluar si la organización espacial de las células difiere entre los grupos estudiados. Para ello utilizamos **Topological Kernel Association Test (TopKAT)**. En este ejemplo, la variable de respuesta corresponde al grupo al que pertenece cada muestra (`FA` o `No FA`) definida anteriormente.


```{r, eval=FALSE}
res <- TopKAT(
  y = y,
  X = NULL,
  K.list = simmat$K.list,
  omega.list = c(0, 0.5, 1),
  outcome.type = "binary"
)
```

### Hipótesis evaluadas

TopKAT contrasta las siguientes hipótesis:

- \*\*Hipótesis nula: la estructura topológica es similar entre los grupos.
- \*\*Hipótesis alternativa: existen diferencias topológicas entre los grupos.

### Valor-p global

El resultado principal del análisis corresponde al valor-p global:

```{r, eval=FALSE}
res$overall.pval
```

De manera general:

- Valores grandes de p (en este caso si (p \geq 0.05) ), no existe evidencia suficiente para rechazar la hipótesis nula.

## Identificación de escalas espaciales relevantes

Hasta este punto hemos evaluado si existen diferencias topológicas entre los grupos. Sin embargo, también es interesante identificar **a qué escala espacial** resaltan esas diferencias.

La función `scale_importance()` permite estudiar cómo cambia la asociación topológica al considerar distintos valores de (\varepsilon). Para ello, el procedimiento recorre una secuencia de escalas y calcula nuevamente la evidencia estadística en cada una de ellas.

```{r, eval=FALSE}
res_scale_import <- scale_importance(
  pd.list = simmat$rips.list,
  y = y,
  omega.list = c(0, 0.5, 1),
  threshold = 50,
  PIDs = 1:100,
  outcome.type = "binary"
)
```

## Conectividad promedio entre grupos FA y NoFA

Hasta este punto hemos explorado muestras individuales. Sin embargo, también resulta útil resumir la organización espacial celular característica de cada grupo clínico y asi podemos ver si existe algún comportamiento en común entre las muestras de un grupo.

Para ello, se construyeron matrices de conectividad promedio para los grupos FA y NoFA utilizando la escala óptima identificada previamente. Estas matrices resumen la frecuencia con la que distintos tipos celulares aparecen conectados dentro de cada condición.

```{r, eval=FALSE}
# ============================================================
# Conectividad promedio FA vs NoFA
# ============================================================

library(dplyr)
library(stringr)
library(reshape2)
library(ggplot2)
library(viridis)

# Un PID por archivo
pid_df <- data1.df %>%
  select(PID, archivo) %>%
  distinct()

# Grupo: FA = 1, NoFA = 0
pid_df$grupo <- ifelse(
  grepl("F", pid_df$archivo, ignore.case = TRUE),
  1,
  0
)

table(pid_df$grupo)

cell.types <- sort(unique(data1.df$type))
threshold <- res_scale_import$min.thresh

# Matrices acumulativas
connect_FA <- matrix(
  0,
  nrow = length(cell.types),
  ncol = length(cell.types),
  dimnames = list(cell.types, cell.types)
)

connect_noFA <- connect_FA

# Calcular conectividad para cada PID
for(i in seq_len(nrow(pid_df))) {

  pid <- pid_df$PID[i]
  grupo <- pid_df$grupo[i]

  patient <- data1.df %>%
    filter(PID == pid)

  connect_i <- generate_connectivity(
    images.df = patient,
    threshold = threshold,
    type.column = "type",
    unique.types = cell.types
  )

  tmp <- matrix(
    0,
    nrow = length(cell.types),
    ncol = length(cell.types),
    dimnames = list(cell.types, cell.types)
  )

  tmp[
    rownames(connect_i),
    colnames(connect_i)
  ] <- connect_i

  if(grupo == 1){
    connect_FA <- connect_FA + tmp
  } else {
    connect_noFA <- connect_noFA + tmp
  }
}

# Promedios
n_FA <- sum(pid_df$grupo == 1)
n_noFA <- sum(pid_df$grupo == 0)

connect_FA <- connect_FA / n_FA
connect_noFA <- connect_noFA / n_noFA

plot_connectivity_matrix <- function(mat, titulo){

  ggplot(
    reshape2::melt(mat),
    aes(Var1, Var2, fill = value)
  ) +
    geom_tile(color = "white") +
    viridis::scale_fill_viridis(option = "turbo") +
    theme_bw() +
    coord_equal() +
    labs(
      x = "Cell Type",
      y = "Cell Type",
      fill = "Average\nConnections",
      title = titulo
    ) +
    theme(
      axis.text.x = element_text(
        angle = 45,
        hjust = 1
      )
    )
}

p_FA <- plot_connectivity_matrix(
  connect_FA,
  paste0("Average Connectivity (FA, n=", n_FA, ")")
)

p_noFA <- plot_connectivity_matrix(
  connect_noFA,
  paste0("Average Connectivity (No FA, n=", n_noFA, ")")
)

p_FA + p_noFA
```

Para cada muestra se construye una matriz de conectividad entre tipos celulares. Posteriormente, las matrices individuales se agrupan según la condición clínica (FA o NoFA) y se promedian para obtener una representación global de cada grupo. Los mapas de calor permiten identificar qué pares de tipos celulares presentan una mayor conectividad promedio.
