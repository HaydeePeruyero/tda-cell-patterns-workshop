# TDA Cell Patterns Workshop

This repository contains the material for a workshop on **Topological Data Analysis (TDA)** applied to the characterization and classification of cellular patterns in epithelial tissue.

The content includes:

* Lecture notes and explanations
* Hands-on exercises
* Code examples (R / Python)
* A reduced dataset for practice

---

## 📚 About this repository

This repository is part of an ongoing research project on the use of TDA for analyzing multicellular organization.

The materials presented here are designed for **educational purposes**, and illustrate the main ideas and methods using a simplified version of the data.

A complete and curated version of the dataset, along with full analyses, will be presented in a forthcoming research article.
